# Stack Produit : Identité, Réputation, Propriété

Le stack produit Fight est construit sur trois couches interconnectées :

- **FightID (Identité)** — identité on-chain portable pour les fans et les combattants
- **Points FP (Réputation)** — points non-transférables qui reflètent la participation
- **$FIGHT (Propriété)** — Jeton d'accès permettant un double accès : projets→fans, fans→expériences

Chaque couche renforce les autres, créant un volant qui stimule l'engagement et l'accumulation de valeur.
